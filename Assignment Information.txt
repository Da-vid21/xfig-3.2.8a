TASK UNDO SHORTCUT

DETAILED INSTRUCTIONS: 
https://github.com/Da-vid21/xfig-3.2.8a/issues/4

GOAL:
The goal of this project is to create a shortcut button that undoes the user's last action.

STEPS:
1. Open Visual Studio Code or your choice of IDE(open it from GitHub).
2. Check if you have opened the whole xfig folder.
3. Click on the search icon on the left toolbar and search "#shortcut". **You should see 3 files with the hashtag.
4. Use the directions provided as comments to work on the assignment.
5. Go to the desktop and compile you code double-clicking the 'compile.sh' file to check if your solution works.

If you are confused at any point with xfig or linux environment, check the youtube video at https://www.youtube.com/watch?v=Csxk165L358.
